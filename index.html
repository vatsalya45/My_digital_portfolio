My Projects

Soil Detection System

The Soil Detection System project analyzed soil quality using sensors and microcontrollers like Arduino to measure moisture, pH, and nutrients. It provided real-time monitoring, automated recommendations, and a user-friendly interface. Challenges like sensor calibration and data filtering were addressed to ensure accuracy. The system improved agricultural efficiency and crop yields, with future plans to integrate machine learning and IoT for enhanced functionality.

Smart Irrigation with Fertilizer Integration system

The Smart Irrigation with Fertilizer Integration system is an automated solution designed to optimize water and nutrient delivery in agricultural fields. It combines irrigation with precise fertilizer and pesticide application, controlled through a MATLAB-based program. The system enhances efficiency by adjusting resources based on real-time soil and environmental data, reducing waste and improving crop yields. This project was also published as a research paper in STM JOURNALS.
